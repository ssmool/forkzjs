## üìå Introduction

**FORKZJS** is a modular JavaScript toolkit for AI-assisted web development. It turns structured data (JSON, XML, RSS, CSV) into dynamic websites and components through simple scripts or terminal commands.

Whether you're a developer, data scientist, or AI engineer, FORKZJS gives your data a web UI instantly.

## üß† Final Script: `run_forkz_sample.sh`

```bash
#!/bin/bash

# -----------------------------
# FORKZJS Auto Runner Script
# -----------------------------
# Generates a site from JSON data,
# extracts ZIP, launches it locally, and opens browser.
# -----------------------------

REPO_URL="https://github.com/ssmool/forkzjs.git"
REPO_DIR="forkzjs"
DATA_FILE="_forkz/forkz_genaix/sample.json_data"
GEN_SCRIPT="forkz_genaiz.sh"
WWW_SCRIPT="forkz_www.sh"
PROJECT_ZIP="project.zip"
PORT=8080
BROWSER_URL="http://localhost:$PORT"

echo "üöÄ Starting FORKZJS Auto Generator..."

# Clone repo if not already
if [ ! -d "$REPO_DIR" ]; then
    echo "üì• Cloning FORKZJS repository..."
    git clone "$REPO_URL"
    if [ $? -ne 0 ]; then
        echo "‚ùå Git clone failed."
        exit 1
    fi
fi

cd "$REPO_DIR" || { echo "‚ùå Failed to enter $REPO_DIR"; exit 1; }

# Make scripts executable
echo "üîß Setting permissions on scripts..."
chmod +x $GEN_SCRIPT $WWW_SCRIPT _forkz_merge.sh

# Check data file
if [ ! -f "$DATA_FILE" ]; then
    echo "‚ùå Data file not found: $DATA_FILE"
    exit 1
fi

# Run generator
echo "üì¶ Generating website from: $DATA_FILE"
echo "$DATA_FILE" | ./$GEN_SCRIPT

# Check if ZIP was created
if [ ! -f "$PROJECT_ZIP" ]; then
    echo "‚ùå Project ZIP not found. Generation may have failed."
    exit 1
fi

# Unzip project
echo "üìÇ Extracting $PROJECT_ZIP..."
unzip -o "$PROJECT_ZIP" -d www_project/

# Run local server
echo "üåê Launching local server at $BROWSER_URL"
./$WWW_SCRIPT &

# Give the server a second to start
sleep 2

# Open browser
echo "üåç Opening browser..."
if command -v xdg-open > /dev/null; then
    xdg-open "$BROWSER_URL"
elif command -v open > /dev/null; then
    open "$BROWSER_URL"
else
    echo "‚ÑπÔ∏è Please open your browser and go to: $BROWSER_URL"
fi

echo "‚úÖ FORKZJS site is now live!"
```
